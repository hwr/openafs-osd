fs wscell

目的

マシンが属しているセルの名前を戻します。

構文 

fs wscell  [-help]

fs ws  [-h]

構文

fs wscell コマンドは、ローカル・マシンのホーム・セルの名前を戻します。

オプション

-help
このコマンドのオンライン・ヘルプを出力します。これ以外の有効なオプションはすべて無視されます。

出力

次の出力は、ローカルの /usr/vice/etc/ThisCell ファイルの内容を、次の形式で表示します。 

   This workstation belongs to cell 'cellname'

例

次の例は、Example Organization セルのマシンに fs wscell が発行されたときの結果です。 

   % fs wscell
   This workstation belongs to cell 'example.org'

必要となる特権

なし。
